Expanded Arbor adds 9 of the overgrown blackplate variants from Portal 2 into Portal Mod. 
Other features include arbored testing elements such as 2 variants of the overgrown super button, 
an overgrown pedestal button, and the overgrown cube dropper.

As of 1.2.0, Expanded Arbor has merged with the mod Portal Mod Deco Blocks, from the Portal Mod Discord. 
This merge has added the following features: 4 new blackplate variants, 2 new lunecast variants, frame tiles, 
and floor debris.
